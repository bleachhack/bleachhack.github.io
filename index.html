About:

BleachHack is an easy to use, free and open source Fabric Utility Mod for Minecraft 1.16.5 and 1.17 Snapshots designed for anarchy servers. BleachHack comes with a clean GUI, a lot of powerful and customizable modules and is frequently updated to have the latest anarchy features, See below for a list of all the features.
